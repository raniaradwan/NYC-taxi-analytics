# NYC Taxi Analytics

## Technologies

- Python
- SQL
- ClickHouse
- FastAPI
- Grafana
- HTML
- JSON

## Project Overview

This project analyzes NYC Taxi trip data using ClickHouse and visualizes KPIs through Grafana dashboards. FastAPI provides REST APIs that return JSON responses to an HTML frontend.

## Features

- ETL Pipeline
- SQL Analytics
- Dashboard Visualization
- REST API
- HTML Frontend
